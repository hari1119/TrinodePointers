Required Field Indicator

Why this ?

Users need to know what are the mandatory fields in a form while filling the field.
UI design should be in a odoo standard.
If the mandatory indicator shows after save, the end user may face inconvenient.
In case of huge fields in a form this indicator helps save time while first filling the manditory fields alone.
